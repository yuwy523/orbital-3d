# Orbital: 3D Solar Explorer

An interactive 3D solar system simulator featuring realistic orbital mechanics and an AI-powered astronomical guide interface.

**Note**: This is a Demo Version. The AI features utilize Mock Data and do not require an API Key to operate.

## Project Structure

- **components/**: React UI components (SolarSystem, Controls, InfoPanel, etc.)
- **services/**: Mock AI Service (Simulated data responses)
- **public/**: Static assets (e.g., textures)
